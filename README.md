# Bulls Season Ticket Cost Calculator [bullstickets.net](https://bullstickets.net)

This project is going to require node and npm and probably some other stuff you may or may not have.

## Local Envirnment

You can use Yarn or NPM
### NPM

run `npm install`

run `next start`



### Yarn

run `yarn install`

run `yarn dev`
